**TL;DR**: Set up a local Kali attacker VM + Juice Shop target, learned HTTP basics and used Burp to intercept requests.

### Environment
- Kali Linux (attacker VM)
- Juice Shop (Docker target)
- Tools: Burp Community, nmap, curl

### Steps I followed
1. Intercepted GET request with Burp and sent to Repeater.  
2. Modified query parameter and observed reflected output.  
3. Saved raw HTTP request (curl command) as PoC.

```
curl -i 'http://10.0.0.42:3000/search?q=test'
```

### Remediation (notes)
- Sanitize and encode output to prevent XSS.
- Use HttpOnly & Secure flags for cookies.
